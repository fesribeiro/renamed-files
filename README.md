# Rename files

This script rename xml/pdf files with base csv file

## Tech
- [Python] - Language programing
- [pip] - Package installer

# Enviroment

#### Configuration:

### Install dependencies:
    pip install -r requirements.txt
    
### Access key
    copy your key access to csv (principal.csv)
    
### Run:
    python rename.py
